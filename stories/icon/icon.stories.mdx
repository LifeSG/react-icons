<Secondary>Overview</Secondary>

A symbol and a visual representative that can be used to
communicate an action or message.

```tsx
import { ArrowDownIcon } from "@lifesg/react-icons/arrow-down";
import { ArrowDownIcon } from "@lifesg/react-icons";
```

<Heading3>Modifying icon style</Heading3>

You can simply customise the icon by extending it and applying your styles

```tsx
import styled from "styled-components";
import { ArrorDownIcon } from "@lifesg/react-icons/arrow-down";

export const MyCustomIcon = styled(ArrowDownIcon)`
    color: red;
    height: 2rem;
    width: 2rem;
`;
```

<Preview>
    <MyCustomIcon />
</Preview>

or if you have a custom css class, you may pass it through the `className` prop

```tsx
<ArrowDownIcon className="my-custom-class" />
```
