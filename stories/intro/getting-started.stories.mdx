<Title>Installation</Title>

Install the LifeSG React icons package.

```
npm i @lifesg/react-icons
```

<Heading3>Usage</Heading3>

```tsx
import { ArrowUpIcon } from "@lifesg/react-icons/arrow-up";

export const Page = () => {
    return <ArrowUpIcon />;
};
```

For more information on the icons and the collection, visit the [documentation](/docs/icon--collection)
